## Overview

SunuPOS is a modern restaurant operating system built to simplify every aspect of restaurant management.

It combines Point of Sale, kitchen workflows, payments, employee management, business analytics and AI-powered assistance into a single platform.

Built with OpenAI coding agents, SunuPOS demonstrates how collaborative AI development can accelerate the creation of production-grade software.

---

## Features

- Point of Sale (POS)
- Kitchen Workflow
- Orders Management
- Employee Management
- Secure Refunds
- Terminal Tracking
- Owner Dashboard
- Business Analytics
- Offline-first architecture
- AI-powered business assistance

---

## Technology

- React Native (Expo)
- NestJS
- Prisma ORM
- PostgreSQL
- TypeScript
- OpenAI API
